## Description
Generates partially filled sudokus (to the extent of your liking) that can be solved.<br/>
Currently used through terminal and not much design is given to how easy or hard the sudokus are to solve.<br/>
Project is under construction and should feature a graphical interface with options for varying options of difficulty.<br/>
<br>
## Usage
Currently only tested on linux. Colour that is added to text uses ANSI escape codes. https://en.wikipedia.org/wiki/ANSI_escape_code<br/>
Simple Makefile is added to run the project.
